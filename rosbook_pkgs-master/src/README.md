## インストール方法
### サンプルプログラムの取得
```bash
cd ~/<catkin_ws>/src
git clone https://github.com/Nishida-Lab/rosbook_pkgs.git
```

### 依存パッケージの解決
```bash
cd ~/<catkin_ws>
sudo rosdep init
rosdep update
rosdep install --from-paths src --ignore-src -r -y
```

### ビルド
```bash
cd ~/<catkin_ws>
catkin_make
source devel/setup.bash
```
